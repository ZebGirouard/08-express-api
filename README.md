# 08 Express API

## Goal
Build a small backend with one `GET` route and one `POST` route so you can inspect request and response flow from the server side.

## Start Here
1. Run `npm install`.
2. Run `npm run dev`.
3. Finish the missing route logic in `server.js`.

The data stays in memory so you can focus on the API shape instead of infrastructure.

## Stretch Goals
- Add a `DELETE` route.
- Validate empty input.
- Save the data to a JSON file.
